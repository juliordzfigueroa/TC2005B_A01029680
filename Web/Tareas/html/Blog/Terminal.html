Comandos de la Terminal

Creado el 25/02/2025

No solo los comandos que usamos en Git son útiles para la terminal, en esta sección se anotarán los comandos que usamos en la terminal para el manejo de archivos y directorios.

Comandos usados en la Terminal

Los siguientes comandos son usados en la terminal de Linux y Mac, y en caso de windows es necesario el instalar Git Bash

Link de la página de Git

Comandos Descripción
pwd Imprime el directorio donde estas trabajando
ls Imprime la lista de archivos (visibles y ocultos) del directorio en donde te encuentras
cd {nombre del directorio} Te cambia al directorio que indiques
cd - Te regresa al directorio anterior del que estas
mkdir {nombre del directorio} Crea un nuevo directorio en el directorio donde se este trabajando
rmdir {nombre del directorio} Borra un directorio vacío
rm -rf {nombre del directorio} Fuerza el borrado de un directorio con todo y su contenido
touch {nombre del archivo} Crea un archivo en el directorio donde se este trabajando, es importante indicar la extensión del archivo
cp {archivo/directorio a copiar} {destino} Copiaun archivo o directorio hacia el destino
mv {archivo/directorio a copiar} {destino} Mueve o renombra un archivo
rm {nombre del archivo} Borra el archivo indicado
cat {nombre del archivo} Muestra en la terminal el contenido del archivo indicado
less {nombre del archivo} Muestra el contenido del archivo página por página
head {nombre del archivo} Muestra las primeras líneas del archivo
tail {nombre del archivo} Muestra las últimas lineas del archivo
grep {patrón} {nombre del archivo} Busca en un archivo líneas que cumplan con el patrón buscado
history Muestra el historial de todos los comandos hechos en la terminal
vi {nombre del archivo} Abré el archivo para ser modificado desde el editor de texto vi
