¿Qué es CSS?

CSS, por sus siglas en ingles Cascading Style Sheets, es un lenguaje de hojas usado para pooder describir las características de un documento HTML. Es tambien uno de los pilares del desarrollo web junto a JavaScript y HTML.

Se compone principalemnte de los siguientes aspectos:

  • Selectores:

    Son la parte del código que indica a que elementos del archivo HTML se le apliacrá cierto estilo. Estos se pueden aplicar a un solo elemento, un grupo de elementos, o a un elementos con una etiqueta especifica:

    La diferencia entre estos es:

    Los selectores de un elemeto especifico se refieren a un tipo especifico ya predefinido en HTML como lo pueden ser los elementos "div" o "p"

    Ejemplo:

    p{

    color: blue;

    }

    Los selectores de clases se deben predefinir en el archivo HTML con el atributo "class". Para poder usarlos en CSS es necesario definirlos con un punto antes.

    Ejemplo:

    .prueba{

    font-weight: bold;

    }

    Por último, para usar un selector por identificador se debe establecer el identificador en el archivo de HTML indicando la sección en la que se requiera el identificador como "id = "nombre_identificador"". Para llamarlo en CSS se debe hacer referencia con el simbolo "#". Como se ve en el siguiente ejemplo.

    Ejemplo:

    #lista_selectores{

    font-size: 24px;

    }

  • Propiedades

    Las propiedades en CSS son las caracteristicas que queremos modificar en los elementos que seleccionamos. Entre las propiedades más comunes se destacan las siguientes:

    • color

      Define el color del texto

    • font-size

      Define el tamaño de la fuente.

    • background-color

      Define el color de fondo de un elemento

    • margin

      Define el espacio exterior alrededor de un elemento

    • padding

      Define el espacio interior de un elemento

  • Valores

    Los valores son los parametros que le asignamos a las propiedades. Estos pueden ser números, nombres de colores o palabras predefinidas en CSS dependiendo de la propiedad que querramos manejar. Aqui unos ejemplos:

    • Valores númericos:

      16, 5, 0

    • Nombres de colores:

      color: blue

    • Clave hexadecimal para el color:

      color: #ff0000

    • Clave RGB para el color:

      color: rgb(255, 0, 0)

    • Unidades de medida para margin, padding, entre otros:

      px (píxeles), % (porcentaje), em

    • Palabras clave:

      center, bold, auto
